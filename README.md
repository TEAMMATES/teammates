TEAMMATES is a free online tool for managing peer evaluations and other feedback paths of your students.

It is provided as a cloud-based service for educators/students and is currently used by hundreds of universities across the world.

<img src="src/web/assets/images/overview.png" width="600">

This is the TEAMMATES GitHub code repository for contributors and developers. **If you are an educator or student looking to use TEAMMATES, visit the [product website](https://teammatesv4.appspot.com).**
